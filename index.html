#TicTacToe Deluxe

Wachsendes Feld · Schieben · Pass & Play
SPIELER A ✕ Siege: 0
SPIELER B ◯ Siege: 0
Spieler A ist am Zug — Feld setzen.

Spielziel

Drei eigene Figuren in einer Reihe (horizontal, vertikal oder diagonal) bringen, egal wie groß das Feld gerade ist.

Setzen

Standard-Zug: eine eigene Figur auf ein leeres Feld stellen.

Schieben

Statt zu setzen, kann man eine gegnerische Figur ein Feld weiterschieben, in alle 8 Richtungen. Das geht nur bei Figuren, an die mindestens eine eigene Figur angrenzt (auch diagonal). Steht im Zielfeld schon eine Figur, wird die auch mitgeschoben, immer weiter, bis eine Figur auf einem freien Feld landet. Läuft die Kette gegen den Rand ohne freies Feld, ist der Zug ungültig.

Bewegen

Eine eigene Figur um ein Feld auf ein freies Nachbarfeld ziehen, ohne Kettenreaktion.

Design-Annahme: diese Aktion war im Original-Konzept als "eigene Fähigkeit" noch offen (Optionen waren u.a. Klonen, Schutz oder Blockieren). Bewegen wurde gewählt, weil es spiegelbildlich zum Schieben ist und sich am einfachsten balancieren lässt. Lässt sich im Code leicht gegen eine andere Variante tauschen.

Wachstum & Blockphase

Nach ein paar Zügen wächst das Feld von 3×3 auf 5×5, bestehende Figuren bleiben stehen. Direkt danach blockiert jeder Spieler nacheinander ein selbst gewähltes Feld: Spieler A markiert seins mit einem Feuer-Icon, Spieler B mit einem Eis-Icon. Steht dort ein Stein, verschwindet er. Blockierte Felder sind dauerhaft raus aus dem Spiel: kein Setzen, Schieben oder Bewegen mehr möglich, und sie zählen nie zu einer Gewinnreihe.

Sonstiges

Der Rahmen ums Spielbrett zeigt, wer gerade dran ist. Der Spielstand (Siege) lebt nur in dieser Sitzung, es gibt keinen Browser-Storage im Vorschaufenster.
